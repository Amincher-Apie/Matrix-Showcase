# Showcase 公开范围说明

本目录只包含适合公开的自研代码和文档。

## 已包含

- Framework 中的自研网络层、逻辑层、事件中心、基础设施和 UI 框架代码
- Scripts 中的 PCG、RunSystem、MissionSystem、InventorySystem、QualityEffects、SO、Excel 工具链、PlayerControl、SpawnSystem、Interaction 等核心脚本
- 项目级文档、模块文档、功能解读文档和 docs 目录下的 Markdown 文档

## 未包含

- 第三方插件完整包
- 模型、贴图、音频、动画、Shader、Prefab、Scene、ScriptableObject 资产
- Unity 生成目录和 IDE 生成文件
- 本机路径依赖和私有版本管理元数据

## 使用方式

请将本仓库视为技术展示材料，而不是完整可运行工程。阅读重点应放在架构、模块边界、网络权威、确定性生成、任务语义和数据驱动设计上。
