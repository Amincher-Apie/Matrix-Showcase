# 小地图使用预烘焙房间缩略图

小地图地形层采用预烘焙俯视缩略图（挂在 PcgRoomRoot 上），CPU SetPixels 渐进拼接，而非运行时俯视相机渲染到 RenderTexture。

**考虑过的替代方案**：运行时架一台正交俯视相机输出到 RenderTexture。这是 Unity 中最直接的做法，无需预处理资产。

**选择预烘焙的原因**：项目追求矢量线性简约风格的小地图表现。俯视相机渲染的是 3D 场景的真实几何，包含光照、阴影、材质细节，成品效果凌乱且不可控。预烘焙允许美术逐房间精确控制缩略图的视觉风格，保证小地图的简洁可读性。

**代价**：每个房间变体需要出俯视缩略图，增加了资产管线维护成本。若后续房间变体数量暴涨，缩略图维护会成为负担。
