# NOTICE

This repository is a technical showcase version of Matrix.

The original Unity project contains third-party assets, commercial plugins, models, textures, audio files, shaders, sample scenes, prefabs, Unity generated folders, IDE generated files, and local development metadata. Those files are intentionally excluded from this showcase repository.

Excluded content includes but is not limited to:

- Unity generated folders: `Library/`, `Temp/`, `Logs/`, `obj/`, `UserSettings/`
- IDE generated files: `.idea/`, `.vs/`, `.vscode/`, `*.csproj`, `*.sln`
- Third-party or commercial packages: Odin/Sirenix, Behavior Designer, Amplify Shader Editor, DOTween, Bullet Impact FX, PolygonDungeon, Protofactor, and similar packages
- Binary assets: models, textures, audio, animation clips, shaders, prefabs, scenes, ScriptableObject assets
- Local machine configuration, local package paths, private version-control metadata

This showcase does not redistribute third-party assets or claim to be a complete playable game project. It is provided only for portfolio, academic review, and technical discussion.
