# Matrix Diagrams

本目录存放 Matrix Showcase 的 Mermaid 架构图。GitHub 可以直接预览 `.mmd` 内容，也可以将其复制到 Mermaid Live Editor 导出 PNG / SVG。

## 图列表

| 文件 | 内容 |
|---|---|
| `architecture-overview.mmd` | 项目总体分层架构 |
| `server-authority-network-flow.mmd` | 服务器权威网络请求链路 |
| `pcg-generation-pipeline.mmd` | 确定性 PCG 地图生成流程 |
| `run-mission-state-flow.mmd` | RunSystem 与 MissionSystem 对局流程 |
| `damage-pipeline.mmd` | 开火、命中、伤害、属性和事件链路 |
| `skill-buff-quality-effects-flow.mmd` | 技能、Buff、品质道具效果管线 |
| `ai-scheduler-lod-flow.mmd` | AI 状态机、感知、调度和 LOD Tick |
| `data-driven-config-flow.mmd` | ScriptableObject 数据驱动配置流 |
